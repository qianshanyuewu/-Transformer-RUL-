## 当前论文总流程

### 第 1 章 绪论

- 明确本文研究目标：从健康状态识别到 Transformer 剩余寿命预测。

### 第 2 章 基于水平振动特征的健康状态识别

本章只做数据与退化表征，不做模型训练。

冻结主线：
1. 读取原始振动数据并以水平振动为主分析对象。
2. 对信号做 `db4 + level=1 + soft threshold` 小波去噪。
3. 提取 `13` 个时域统计特征作为候选特征。
4. 对候选特征做 SG 平滑与评分分析。
5. 固定得到剔除 `mean_abs` 后的 `12` 个累积建模特征。
6. 构建融合健康指标。
7. 用 `3σ` 规则定位 FPT。

### 第 3 章 基于 Transformer 的 RUL 预测模型

冻结主线：
1. 输入：第2章确定的 `12` 个水平振动累积时域特征。
2. 标签：寿命比值 `P`。
3. 切窗：全寿命序列，`window_size = 10`，`step_size = 1`。
4. 主模型：编码器-解码器 Transformer。
5. 对比模型：LSTM、GRU。
6. 先做一轮有边界的人工敏感调参，再冻结正式基线。

### 第 4 章 Transformer 自动调参与结果分析

冻结主线：
1. 仅对 Transformer 做 Optuna 自动调参。
2. 搜索阶段只使用训练集与验证集。
3. 测试集只在最优参数冻结后评估一次。
4. 分析人工调参与自动调参的差异及改进幅度。

### 第 5 章 结论与展望

- 总结从健康状态识别到 RUL 预测的完整流程。
- 总结 Transformer 相对 LSTM、GRU 的表现。
- 讨论后续在更多传感器、多工况迁移与领域自适应方向上的扩展空间。
