# 第2章 基于水平振动特征的健康状态识别

## 2.1 数据集与研究对象

本文以 XJTU-SY 滚动轴承加速寿命试验数据集为研究对象。该数据集包含 `35Hz12kN`、`37.5Hz11kN` 和 `40Hz10kN` 三种工况，每种工况下各有 `5` 个滚动轴承，共 `15` 个轴承全寿命退化样本。原始数据以 `csv` 文件形式存储，每个采样文件按时间顺序命名为 `1.csv`、`2.csv` 至 `N.csv`，单个文件包含 `32768` 个采样点，其中第一列为水平振动信号，第二列为垂直振动信号。

为了保证第2章和第3章的建模口径一致，本文在最终冻结协议中统一采用**水平振动信号**作为主分析对象。这样既保留了数据集中最直接、最稳定的退化观测通道，也避免了“第2章分析一套特征、第3章建模另一套特征”的问题。

## 2.2 水平振动信号预处理与候选特征提取

原始振动信号在采集过程中不可避免地受到环境噪声和测量干扰影响。若直接在原始信号上提取统计特征，容易将高频随机波动一并带入退化表征过程。因此，本文在候选特征提取之前，先对水平振动信号执行小波阈值去噪。

当前冻结参数为 `db4 + level=1 + soft threshold`。选择 `db4` 小波的原因在于，Daubechies-4 具有较好的时频局部化能力，能够有效保留轴承故障信号中的局部冲击信息；一级分解既能削弱随机高频扰动，又不会过度抹平早期退化细节；软阈值方式则有利于获得连续、平滑的重构信号。去噪阈值采用通用阈值（universal threshold），即 $\lambda = \hat{\sigma} \sqrt{2 \ln N}$，其中 $\hat{\sigma} = \text{median}(|d_1|) / 0.6745$ 为噪声标准差的鲁棒估计，$d_1$ 为一级小波细节系数，$N$ 为信号长度。

在完成去噪后，本文从每个采样时刻的水平振动信号中提取 `13` 个时域统计特征作为候选特征。设某一采样时刻的去噪信号为 $x = \{x_1, x_2, \ldots, x_N\}$，各特征定义如下：

| 序号 | 特征名称 | 计算公式 |
|:---:|:---:|:---:|
| 1 | 反正切标准差 | $\text{Std}(\arctan(x_i))$ |
| 2 | 反双曲正弦标准差 | $\text{Std}(\text{arcsinh}(x_i))$ |
| 3 | 标准差 | $\sigma = \sqrt{\frac{1}{N}\sum_{i=1}^{N}(x_i - \bar{x})^2}$ |
| 4 | 峰峰值 | $x_{\max} - x_{\min}$ |
| 5 | 均方根 | $\text{RMS} = \sqrt{\frac{1}{N}\sum_{i=1}^{N}x_i^2}$ |
| 6 | 上边界 | $x_{\max} + \frac{x_{\max} - x_{\min}}{2(N-1)}$ |
| 7 | 冲击因子 | $\max|x_i| \;/\; \overline{|x|}$ |
| 8 | 峰值因子 | $\max|x_i| \;/\; \text{RMS}$ |
| 9 | 裕度因子 | $\max|x_i| \;/\; \left(\frac{1}{N}\sum\sqrt{|x_i|}\right)^2$ |
| 10 | 能量 | $E = \sum_{i=1}^{N}x_i^2$ |
| 11 | 峭度 | $K = \frac{1}{N}\sum_{i=1}^{N}\left(\frac{x_i - \bar{x}}{\sigma}\right)^4$ |
| 12 | 平均绝对值 | $\overline{|x|} = \frac{1}{N}\sum_{i=1}^{N}|x_i|$ |
| 13 | 偏度 | $S = \frac{1}{N}\sum_{i=1}^{N}\left(\frac{x_i - \bar{x}}{\sigma}\right)^3$ |

其中，反正切标准差和反双曲正弦标准差分别通过 $\arctan$ 和 $\text{arcsinh}$ 变换对原始信号进行非线性压缩后计算标准差，能够在抑制极端振幅影响的同时保留退化趋势信息。

## 2.3 候选特征评分与固定12特征确定

候选特征提取完成后，本文先采用 Savitzky-Golay 滤波器对各条特征序列进行平滑处理，以降低局部抖动对退化趋势判断的影响。随后，采用”单调性 + 时间相关性”联合评分方法，对各候选特征的趋势表达能力进行量化比较。

需要强调的是，在当前冻结协议中，这一步只作为候选特征排序参考，而不再动态筛出一套因工况变化而改变的“最优特征子集”。第2章与第3章统一采用同一套固定建模特征，即在 `13` 个候选特征中仅剔除 `mean_abs`，保留其余 `12` 个累积时域特征用于后续建模。

之所以单独剔除 `mean_abs`，是因为该特征在累积变换后与同族幅值特征存在较强重复性，且对后续寿命预测贡献不够稳定。因此，第2章的输出形式不再是“动态筛出的 13 个关键特征”，而是“13 个候选特征评分 + 固定 12 个建模特征”。

## 2.4 健康指标构建与 FPT 检测

在固定 `12` 个特征后，本文进一步对每条特征序列执行累积变换，以增强长期退化趋势。设第 $j$ 个特征在时刻 $t$ 的原始值为 $f_j(t)$，累积变换定义为：

$$
\tilde{f}_j(t) = \frac{\sum_{\tau=1}^{t} f_j(\tau)}{\sqrt{\sum_{\tau=1}^{t} f_j(\tau)^2}}
$$

该变换通过累积求和与能量归一化，将瞬时波动转化为单调递增的趋势表示。

随后，构建多特征融合健康指标。以寿命前 $B$ 个采样点（本文取 $B=10$）作为健康基线，计算各特征在基线窗口内的均值 $\mu_j$ 和标准差 $\sigma_j$。对每个时刻 $t$，计算各特征相对于基线的标准化偏离并取平均：

$$
HI(t) = \frac{1}{D} \sum_{j=1}^{D} \left| \frac{\tilde{f}_j(t) - \mu_j}{\sigma_j} \right|
$$

其中 $D=12$ 为特征维度。$HI(t)$ 在健康阶段保持较低水平，当退化发生时将显著上升。选取前 10 个采样点作为基线窗口，是因为滚动轴承在服役初期通常处于稳定运行状态，该范围足以建立可靠的健康基准，同时避免纳入可能已出现早期退化的样本。

为定位轴承从健康阶段进入稳定退化阶段的时间点，本文采用 `3σ` 准则自动检测首次预测时间（First Predicting Time, FPT）。以前 $B$ 个采样点的健康指标为基线，阈值定义为 $\theta = \bar{HI}_B + 3 \cdot \text{Std}(HI_B)$，其中 $\bar{HI}_B$ 和 $\text{Std}(HI_B)$ 分别为基线窗口内健康指标的均值和标准差。当健康指标在基线之后连续 `3` 个采样点超过阈值 $\theta$ 时，将首次越界位置定义为 FPT。采用 `3σ` 准则是因为该方法在正态分布假设下对应 99.7% 的置信水平，能够有效区分正常波动与真实退化信号。

按当前冻结协议在三工况全量数据上的重建结果表明，`15` 个轴承均检测到明确的 FPT，FPT 索引范围为 `14` 至 `17`，平均值为 `14.93`。需要说明的是，本章保留 FPT 作为退化阶段识别结果，用于验证健康指标的有效性和退化检测能力。在第3章的 RUL 预测建模中，并不以 FPT 作为样本切窗的起点，而是基于全寿命序列构造训练样本。这一设计是为了保留完整的寿命退化信息，避免因截断导致的信息损失，同时使不同轴承的标签空间保持一致。

## 2.5 本章小结

本章面向 XJTU-SY 原始振动数据，构建了与后续建模完全一致的健康状态识别流程。首先，对水平振动信号执行 `db4 + level=1 + soft threshold` 小波去噪；其次，提取 `13` 个时域统计特征并进行 SG 平滑与评分分析；随后，冻结得到剔除 `mean_abs` 后的 `12` 个累积建模特征；最后，通过融合健康指标和 `3σ` 准则完成 FPT 检测。

全量重建结果表明，当前协议能够在 `15/15` 个轴承上稳定识别 FPT，FPT 索引集中在 14~17 范围内（平均 14.93），表明所构建的融合健康指标具备对退化起始阶段的敏感识别能力。此外，各工况下的健康指标曲线均呈现出从稳定期到加速退化期的清晰过渡（如图2-7所示），验证了累积变换和多特征融合策略对退化趋势增强的有效性。

本章的特征提取与健康指标构建流程为第3章的 Transformer、LSTM 和 GRU 寿命预测模型提供了统一、可复现的输入基础。需要指出的是，FPT 检测结果在本章中主要用于验证特征体系的退化表征能力；在第3章的建模实验中，模型将基于全寿命序列构造训练样本，以充分利用完整的退化信息。
