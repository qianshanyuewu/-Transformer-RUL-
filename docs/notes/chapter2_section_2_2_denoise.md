# 2.2 水平振动信号去噪

原始振动信号在采集过程中不可避免地受到环境扰动和测量噪声影响，若直接在原始序列上提取统计特征，容易放大高频随机波动对退化表征的干扰。综述文献指出，小波阈值去噪是滚动轴承信号处理中较为常见且有效的信号重构降噪方法；相关 Transformer 文献也采用了“先去噪、再提取统计特征”的处理路径。因此，本文在候选特征提取之前，先对原始水平振动信号进行小波去噪。

具体而言，本文将每个 `csv` 文件中的水平振动序列视为单条一维输入，采用 `db4` 小波基进行阈值去噪。去噪模块的输入对象为单条一维振动序列，输出对象为与输入长度一致的重构序列，因此该步骤不会改变单次采样的时间长度，也不会破坏后续按采样时序提取特征的流程。

在参数选择上，本文没有直接沿用旧版本中的四层小波去噪设置，而是先对不同层数和阈值方式进行了客观对照。检查结果表明，原先的 `level=4 + soft threshold` 会对原始波形造成明显压缩：在 15 个轴承首个采样文件共 30 条通道序列上的平均相关系数仅为 `0.555246`，平均标准差保留比例仅为 `0.476363`，说明该设置存在过度去噪问题。相比之下，`level=1 + soft threshold` 在抑制高频成分的同时，对原始波形保留更好，其平均相关系数为 `0.879759`，平均标准差保留比例为 `0.878717`。因此，本文当前将 `db4 + level=1 + soft threshold` 作为第2章的去噪起点参数。

基于这一设置，去噪后的水平振动序列既保持了原始退化波形的主要结构，又能够有效降低高频随机成分的占比，为后续时域统计特征提取提供更稳定的输入基础。
