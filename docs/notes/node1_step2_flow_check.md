## 前检查

当前第2章约定主线为：

1. 读取 XJTU-SY 原始双通道振动。
2. 小波去噪。
3. 提取时域与频域候选特征。
4. 对特征序列平滑。
5. 用单调性与时间相关性做辅助排序。
6. 用 Pearson 相关系数做去冗余。
7. 构建融合健康指标。
8. 用 `3σ` 自动检测 FPT。

## 实际运行流程核对

### 已与当前要求一致的部分

1. 当前流程只读取双通道振动，不使用温度。
2. 当前流程对每条信号执行 `db4` 四层小波去噪。
3. 当前流程按每个通道 `13` 个时域特征和 `6` 个频域特征提取候选特征，共 `38` 维。
4. 当前流程对每条特征序列执行 SG 平滑。
5. 当前流程使用“单调性 + 时间相关性”做辅助排序，而不是直接当最终结论。
6. 当前流程使用跨轴承平均 Pearson 做去冗余，而不是把不同轴承直接拼接后再计算。
7. 当前流程在筛选后执行累积变换，再构建融合健康指标。
8. 当前流程最终使用 `3σ` 规则检测 FPT。

### 尚未冻结、只能视为临时实现的部分

1. 当前特征评分和去冗余只基于 `train bearings` 进行，这已经提前带入了第3章训练/测试划分思想。
2. 当前 `top_k_features = 12` 是人为上限，不是文献或毕设要求直接给出的值。
3. 当前单调性与时间相关性的权重固定为 `0.5 / 0.5`，这只是临时设定。
4. 当前 Pearson 阈值 `0.90` 虽然方向上比旧版合理，但仍属于待论证参数。
5. 当前累积变换公式和健康指标融合方式已经可运行，但本质上仍是“当前选定实现”，还没有被正式冻结为论文唯一表述。
