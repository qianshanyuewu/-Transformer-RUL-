## 1. 基于 Transformer 模型的滚动轴承剩余使用寿命预测方法

文件：
- `文献/基于Transformer模型的滚动轴承剩余使用寿命预测方法 (1).pdf`

提取到的关键流程：
- 从原始振动信号出发，先做离散小波去噪。
- 在去噪信号上提取统计特征。
- 对特征做 SG 平滑，以减少波动。
- 通过累积变换增强单调性与趋势性。
- 通过单调性与趋势性指标筛选特征。
- 采用 Transformer 进行 RUL 预测。

对当前重建有用的点：
- 证明“去噪 -> 特征提取 -> 平滑 -> 趋势增强 -> 特征筛选 -> 时序预测”是一条完整可行的路线。
- 累积变换可以作为增强退化趋势的候选步骤。
- 时序建模阶段可以借助 Transformer 捕捉长期依赖关系。

对当前重建不直接照搬的点：
- 该文主要围绕 PHM2012 数据集展开，不是 XJTU-SY。
- 该文使用寿命比值 `P` 作为标签，并采用编码器-解码器 Transformer。
- 当前毕设要求更强调直接预测剩余循环次数，并在 XJTU-SY 上对比 LSTM、GRU。

## 2. 滚动轴承剩余使用寿命预测综述

文件：
- `文献/滚动轴承剩余使用寿命预测综述.pdf`

提取到的关键认识：
- RUL 预测通常包含：降噪、特征提取、特征选择/降维、特征融合、退化指标构建、预测建模。
- 小波降噪是常见且合理的信号预处理方式。
- 退化特征与故障诊断特征的评价重点不同：
  - 故障诊断强调可分性。
  - 性能退化与 RUL 预测更强调单调性、趋势性、稳定性。
- 多域特征能够更全面地刻画退化信息，但冗余特征会降低模型效果。
- 健康指标或退化指标通常是多特征融合结果，而不是单一统计量。
- RUL 评价方法中，误差指标和过早/滞后预测惩罚都很重要。

对当前重建有用的点：
- 可以支持我们把第2章写成“特征提取 + 特征筛选 + 融合指标构建”的结构。
- 可以支持我们在第3章同时采用 MAE、RMSE、MAPE 评价模型。
- 可以支持“先构建健康状态识别，再做 RUL 定量预测”的完整流程。

## 4. 基于深度学习的轴承使用寿命的预测研究

文件：
- `文献/aam_2624708（1）.pdf`

提取到的关键认识：
- 深度学习模型中，时序建模对 RUL 预测非常关键。
- LSTM 能有效捕捉序列依赖，CNN 更擅长提取局部模式。
- CNN + LSTM 的融合模型通常优于单独 CNN 或单独 LSTM。
- 该文给出了用 MAE 和得分指标评价预测结果的思路。

对当前重建有用的点：
- 可以支持我们把 LSTM、GRU 作为对比基线。
- 可以支持“时序特征是寿命预测核心”的论点。
- 说明加入局部特征提取模块是可讨论的后续增强方向。
