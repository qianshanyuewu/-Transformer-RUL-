# 2.3 13维时域候选特征提取

在完成水平振动信号去噪后，需要从每个采样时刻的信号中提取能够表征退化演化的统计特征。当前冻结协议直接围绕第3章最终建模所需的时域统计量构建候选集，不再扩展到双通道或时频混合的大候选池。

本文固定提取 `13` 个时域候选特征：反正切标准差、反双曲正弦标准差、标准差、峰峰值、均方根、上边界、冲击因子、峰值因子、裕度因子、能量、峭度、平均绝对值和偏度。该特征集覆盖了幅值尺度、极值范围、冲击性与分布形状等多个退化描述维度。

之所以采用这组紧凑的候选特征，是因为它们与后续寿命预测任务的关系最直接，也更容易与第3章的固定建模特征保持一致。与早期的大候选池相比，当前方案减少了输入分歧，有利于建立统一的方法链。

对当前候选特征提取模块的检查表明，其输入对象为单条去噪后的一维水平振动序列，输出对象为 `13` 维候选特征向量。对 15 个轴承首个采样文件的复核未发现 `NaN` 或 `Inf`，说明当前候选特征池在数值上是稳定的。基于这一结果，本文将以这 `13` 个时域候选特征作为后续平滑、评分和健康指标构建的输入基础。
