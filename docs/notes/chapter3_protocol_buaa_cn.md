# 第3章建模协议（北航论文式固定12特征版）

## 1. 当前协议定义

在 `thesis_rebuild` 中，`buaa_paper` 协议已冻结为：

1. 仅使用水平振动信号
2. 候选特征为 `13` 个时域统计特征
3. 最终建模特征为剔除 `mean_abs` 后的 `12` 个累积时域特征
4. 标签采用寿命比值 `P`
5. 使用全寿命序列切窗
6. `window_size = 10`
7. `step_size = 1`
8. `use_fpt_start = false`
9. 特征缩放模式为 `minmax`
10. decoder 起始值固定为 `1.0`

## 2. 与第2章的关系

第2章与第3章当前已经统一为同一套固定12特征协议：

1. 第2章负责信号预处理、候选特征评分、固定12特征确定、HI 与 FPT
2. 第3章直接使用这 `12` 个累积特征进行建模
3. FPT 只保留为第2章退化识别结果，不作为第3章默认切窗起点

## 3. 与第4章的关系

第4章只在该协议之上对 Transformer 做自动调参，不改变：

1. 输入特征集合
2. 数据划分
3. 标签定义
4. 基本评价指标

因此，第4章优化结果可以直接和第3章人工调参结果对比。
