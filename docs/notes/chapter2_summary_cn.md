# 第二章重建摘要

## 当前冻结口径

- 数据集：XJTU-SY 滚动轴承加速寿命试验数据集
- 数据源：仅使用振动数据
- 主分析对象：水平振动信号
- 协议名：`buaa_paper`

## 当前方法链

1. 按采样顺序读取原始振动信号。
2. 对水平振动信号执行 `db4 + level=1 + soft threshold` 小波去噪。
3. 提取 `13` 个时域统计特征作为候选特征。
4. 对每条候选特征序列执行 SG 平滑。
5. 计算单调性与时间相关性评分，仅作为排序参考。
6. 固定保留剔除 `mean_abs` 后的 `12` 个累积时域特征。
7. 对保留特征执行累积变换。
8. 构建多特征融合健康指标。
9. 用 `3σ` 准则自动检测 FPT。
