# 2.5 多特征融合健康指标与 FPT 检测

在完成候选特征筛选之后，本文进一步对保留特征执行累积变换，以增强退化趋势。这里采用的累积变换不是简单求和，而是逐点进行“累积和 / 累积平方和开根号”的缩放处理。其目的在于一方面保留随寿命推进的累积效应，另一方面抑制不同量纲和幅值水平对趋势表达的影响。代表特征序列上的检查结果表明，累积变换后特征的单调性和时间相关性均有明显提升，因此该步骤在当前第2章中可以保留。

在累积变换基础上，本文采用多特征融合方式构建健康指标。具体而言，以寿命早期的若干采样点作为健康基线，对每个时刻多维特征相对基线均值的标准化偏离进行计算，并在特征维上取平均，得到单一健康指标序列。该指标的含义是：轴承当前状态相对健康基线偏离越明显，其值越大。与直接使用单一特征相比，这种方式能够同时利用固定 `12` 个累积时域特征的互补信息，更适合作为第2章中的退化表征结果。

为定位轴承从健康阶段进入稳定退化阶段的起点，本文采用 `3σ` 准则进行 FPT 检测。具体规则为：以前 `10` 个采样点的健康指标作为基线，计算其均值与标准差，并构造 `mean + 3σ` 阈值；当健康指标在基线之后连续 `3` 个采样点超过该阈值时，将首次越界位置定义为 FPT。按当前冻结协议在三工况全量数据上的重建结果表明，`15` 个轴承均检测到明确的 FPT，FPT 索引范围为 `14` 至 `17`，平均值为 `14.93`。

基于上述结果，本文当前将“累积变换 + 多特征融合健康指标 + `3σ` 自动检测 FPT”作为第2章退化起点识别的正式方法链。需要说明的是，FPT 在本章中用于刻画退化阶段起点，但不再作为第3章默认的切窗起点。
